# ailib.info 生态重构总体规划

## 📋 项目背景

### 现状
- **原有定位**: ai-lib (单一 Rust SDK) 介绍网站
- **覆盖范围**: 仅 ai-lib 和 ai-lib-pro（已废止）
- **目标受众**: Rust 开发者，企业客户
- **内容结构**: 单一产品的营销 + 文档

### 问题
1. ❌ 内容过时 - 引用已废止的 ai-lib-pro
2. ❌ 定位局限 - 未反映新的三项目生态
3. ❌ 导航混乱 - 项目关系不清晰
4. ❌ 内容缺失 - 无 ai-protocol 和 ai-lib-python 介绍

## 🎯 重构目标

### 新定位
**ailib.info** - AI-Protocol 生态系统官方门户
展示统一的 AI 模型交互规范和多语言运行时实现

### 新目标受众
1. **项目决策者** - 选择合适的运行时
2. **协议 implementers** - 基于 AI-Protocol 实现新 runtime
3. **开发者** - 使用 Rust 或 Python runtime
4. **企业团队** - 生产级部署和集成
5. **研究者** - 了解协议设计和架构

### 三项目生态
```
┌─────────────────────────────────────┐
│        AI-Protocol (v0.4.0)         │  ← 规范层
│   Data-State Rulebook + Operators   │

              │
    ┌─────────┴─────────┐
    │                   │
┌───▼──────┐      ┌────▼────────┐
│ ai-lib   │      │ ai-lib      │  ← 实现层
│  -rust   │      │  -python    │
│(v0.6.6)  │      │  (v0.5.0)   │
└──────────┘      └─────────────┘
```

```
- 配置示例（YAML/JSON 示例代码）
- 文档快速链接→ ~/ai-protocol/docs/
```

### Rust SDK 专区 （/rust）
```
- 产品概览（性能、特性和优势）

- 文档入口（链接到 README）
- GitHub 仓库链接
```

### Python SDK 专区 （/python）
```
- 产品概览（PyPI v0.5.0 已发布）
- 快速开始（pip install + 示例）
- 核心功能与 Rust 对比
- 文档入口（链接到 README）
- PyPI 仓库链接
```

### Provider 目录 （/providers）
```
分为三大类：
1. Global Providers (OpenAI, Anthropic, Groq, ...)
2. China Region (百川、通义、智谱、月之暗面、...)
3. Local/Custom (Ollama, vLLM, 自己托管)

每项属性：
- Logo
- Provider Name
- 支持的模型
- 协议状态（stable/partial）
```
